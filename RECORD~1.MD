# Recording checklist — Lebanese audio for Hkeeli

Record each word/phrase in your own voice (or a family member's), save each one
as an **.m4a** file with the EXACT filename shown, and upload them all into an
**audio/** folder in the GitHub repo. The app automatically plays your recording
when it exists, and falls back to robot-Arabic text-to-speech when it doesn't —
so you can record in batches, in any order.

**Easiest workflow on iPhone:** Voice Memos → record → rename the memo to the
filename (without .m4a) → Share → Save to Files → then upload to GitHub from a
computer (or the GitHub website in Safari). Voice Memos saves .m4a natively.

**Tip:** say each word twice in one recording — once at normal speed, once slowly.
