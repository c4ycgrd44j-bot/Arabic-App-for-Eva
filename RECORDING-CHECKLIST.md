# Recording checklist — Lebanese audio for Hkeeli

Record each word/phrase in your own voice, save as **.m4a** with the EXACT
filename shown, and upload into an **audio/** folder in the GitHub repo.
The app plays your recording when it exists and falls back to robot text-to-speech
when it does not — record in batches, any order.

**iPhone workflow:** Voice Memos → record → rename memo to the filename (without .m4a)
→ Share → Save to Files → upload to GitHub. Voice Memos saves .m4a natively.

**Tip:** say each word twice in one recording — once normally, once slowly.
For words with two forms (kifak / kifik), record both in the same file.
